# CSS Selectors and Filters

```css
/* only get the first element of css-selector*/
css-selector:first

/* only get the last element */
css-selector:last

/* only apply to even index element */
css-selector:even

/* only apply to odd index element */
css-selector:odd

/* only apply to first child of css-selector */
css-selector:first-child

/* only apply to the last css-selector */
css-selector:last-of-type

css-selector:parent

p:contains('specific word')
```
