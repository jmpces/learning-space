# **Ubuntu**

- ## **Installing the SDK**

    [Link Reference](https://learn.microsoft.com/en-us/dotnet/core/install/linux-ubuntu#install-the-sdk)

    ```
    sudo apt-get update
    sudo apt-get install -y dotnet6
    ```

- ## **Installing only the Runtime**

    ```
    sudo apt-get update
    sudo apt-get install -y aspnetcore-runtime-6.0
    ```
